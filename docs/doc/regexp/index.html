      # 匹配字符

      字符含义
      \s | \S空格 | 非空格
      $开始
      ^结束
      ()匹配子串不能携带g参数
      *匹配0或多次
      +至少一次
      ?最多一次
      .代表除换行外的任意字符
      {}限定个数{5} {5,10} {5,}
      []罗列匹配范围[0-9][a-z][A-Z] [^xyz] [ \f\r\t\n][\u4e00-\u9fa5]
      \w | \W合法字符 | 非法字符
      \b单词边界
